BuildingsÂ¶

A description of the different buildings can be found here. The models Apartments2Thermal-v0 and Apartments2Grid-v0, ApartmentsThermal-v0 and ApartmentsGrid-v0, and SeminarcenterThermostat-v0 and SeminarcenterFull-v0 are based on the same buildings respectively, but differ in the controllability.

For each model we define an evaluation scenario that includes the tracked KPIs, the evaluation weather file and the evaluation interval. These fixed scenarios should be used to evaluate the control performance on a specific building model.
