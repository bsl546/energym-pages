A class to track the KPIs while running simulations.

The KPIs are customizable by specifying them in the kpi_options dict. An example for the dict looks as follows:

âkpi1â:{ânameâ:âZ01_Tâ ,âtypeâ:âavg_devâ, âtargetâ:[19,23]},

âkpi3â:{ânameâ:âFa_Pw_Allâ,âtypeâ:âavgâ},
